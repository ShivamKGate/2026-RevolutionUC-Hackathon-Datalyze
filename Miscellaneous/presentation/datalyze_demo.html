Problem Statement

  • SMEs and nonprofits lack access to enterprise analytics
  • Data is siloed across departments
  • Manual reporting wastes 15–20 hours per week
  • No actionable insights from raw data

Solution — Datalyze

  • Upload any data (CSV, Excel, PDF, JSON, images)
  • AI classifies the analysis track automatically
  • 21 specialized AI agents analyze in parallel
  • Rich interactive visualizations + knowledge graphs
  • Export professional PDF reports

Tech Stack Deep Dive

  • Frontend: React + Vite + TypeScript + Plotly + D3 + Force Graph
  • Backend: FastAPI + CrewAI + SQLAlchemy + PostgreSQL + pgvector
  • AI: Ollama (local LLMs) + Featherless (cloud LLMs) + Gemini (classification) + ElevenLabs (narration)
  • Agents: 21 specialized AI agents with structured JSON contracts

Architecture

Flow with DAG-based parallel execution and stage gates between phases.

User → Upload → File Classifier → Pipeline Classifier → Orchestrator
Stage gate · classify
Smart Agent Selection → Agents (parallel DAG) → Aggregation → Visualization
Parallel DAG · only relevant agents per track

Agent Specialization

  • Each agent: unique role, strict JSON schema, confidence scoring
  • Categories: Processors (5), Analyzers (6), Synthesizers (4), Utilities (6)
  • Normalizer bridges agent output to orchestrator contracts
  • Smart routing: only relevant agents run per track

Analysis Tracks

  • Predictive Analytics: time-series forecasting, anomaly detection, driver decomposition
  • Automation Strategy: process bottleneck analysis, ROI calculation, SOP generation
